# 🖼️Textures

All textures are stored in the `textures` folder. In this folder, you will find two new folders named "blocks" and "tools".&#x20;

* `blocks` folder: Stores the texture files for all archaeology blocks.&#x20;
* `tools` folder: Stores the texture files for all archaeology tools.&#x20;

You need to place the corresponding textures in their respective folders. The images should be in PNG format, and the file names are also be used as the texture ID (without the file extension).

The texture used for each archaeology block is determined by the `stages` section in their own configuration. For more info, please view [Archeology Block](archeology-block.md) page.
