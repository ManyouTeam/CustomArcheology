# ⌨️Commands

## Archify

Usage: /customarcheology archify \<BlockID> \<World>

Description: Make specified world start generate Archeology Block.

## Dearchify

Usage: /customarcheology dearchify \<BlockID> \<World>

Description: Make specified world no longer generate Archeology Block.

## Give

Usage: /customarcheology give block/tool \<BlockID/ItemID> \<Amount>

Description: Give player items that registered by CustomArcheology

## Reload

Usage: /customarcheology reload

Description: Reload the plugin.

## Place

Usage: /customarcheology place \<BlockID> \<World> \<x> \<y> \<z>

Description: Directly place archeology block at specified location.

## Generate Item Format

Usage: /customarcheology generateitemformat.

Description: Generate item format for hold item.
