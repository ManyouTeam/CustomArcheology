# 🔑Permissions

## Use Command

Require `customarcheology.command.<subCommand>`, like customarcheology.command.reload for /ca reload command.

## Excavate Blocks

Require both `customarcheology.play.archeology` and `customarcheology.play.archeology.<blockID>`, like `customarcheology.play.archeology.suspicious_deepslate`  for `excavate suspicious_deepslate` block.

## Highlight

Require `customarcheology.play.highlight` to highlight near archeology block.
