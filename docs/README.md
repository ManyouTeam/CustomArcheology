## Info

* CustomArcheology is a Spigot plugin for custom archaeology.
* This plugin adds custom archaeology tools, blocks, and loot tables to your server.
* It utilizes resource packs to provide custom textures and animations for the archaeology tools and blocks.
* The loot tables support third-party plugin items like MMOItems, Eco-style, ItemsAdder, Oraxen and can execute commands or spawn monsters by using action system.
* It allows for the random generation of archaeology blocks in the world, with support for multiple generation rules and limits such as biome and height.
* Can generate archeology block at loaded (old) chunks!
* The items displayed to players when excavating archaeology blocks can differ from the actual items obtained, ensuring anti-cheat measures or creating deceptive effects.
* Custom language files are supported, allowing for the customization of all message contents.
* New! Support auto generate archeology block at vanilla structure or BetterStrucutres strucutre!
* Your archeology block can rollback to the original replaced block if you delete CustomArcheology plugin! (Require you use /stop command in server and then delete the pluginm, still in test)
